Dev Projects

  • Multiple Personality Bot: Making a Discord and telegram bot that can impersonate any famous celebrity/figure, past or present
  • Personal assistant app: The idea is I want one console-like app that can do everything I need. Anytime I get an idea, note something I need to buy, or want to remember something, I want to be able to do it in one place. I want to be able to ask it questions, and have it either just record the data in the appropriate place or database, or potentially take action on it. Some things it's doing already:
    • Recording ideas to an ideas database
    • Recording groceries to buy, chores to do, and filing them appropriately
  • Spotify Companion: App/website that can improve on the base functionality. Some goals are controls to instantly hate the song playing now, and ensure it will never be played again, getting a better discovery mode that can both explore certain genres and making sure to only play music you have not heard before.
  • TLDR Bot: Another Discord and Telegram bot that can summarize what you've missed in a channel since you've been gone. It uses OpenAI API to take the conversation, break it into smaller pieces (within the token limit), summarize each of those pieces, and then make a full summary of all those summaries. It's a work in progress, but it's getting better.
